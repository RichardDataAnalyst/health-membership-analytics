# 🩺 Health Membership Analytics Tool

**Portfolio Project by Richard**  
Medical Student & Aspiring Health Data Analyst

## Overview
A complete Python application for managing health club members with integrated clinical analytics. I combined object-oriented programming with real medical knowledge to track membership, calculate BMI, monitor Resting Heart Rate, and assess cardiovascular risk.

## Features
- Object-oriented membership system with automatic expiry dates
- Clinical metrics integration: BMI classification and Resting Heart Rate (RHR) monitoring
- Data cleaning and analysis with Pandas
- Interactive visual dashboard (Matplotlib + Seaborn)

## Tech Stack
- Python
- Pandas & NumPy
- Matplotlib & Seaborn

## The Clinical Edge
As a medical student, I added real clinical logic (BMI categories + RHR risk assessment) instead of generic metrics. This makes the tool directly relevant for health and wellness applications.
